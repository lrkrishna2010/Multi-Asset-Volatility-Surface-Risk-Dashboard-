# Multi-Asset Volatility Surface & Risk Dashboard 

This project calculates **volatility surfaces** for multiple assets, simulates **portfolio risk**, and provides a **dashboard** for visualizing risk metrics and Greeks.

---

## Features
- Market data ingestion and preprocessing
- Implied volatility calculation for options
- Volatility surface construction and 3D visualization
- Monte Carlo simulations for portfolio risk
- VaR, CVaR, and Greeks calculations
- Interactive dashboard with `plotly`
